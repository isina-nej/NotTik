# Categories and Search Specification

## Purpose
Help users quickly find specific notifications within a large local dataset.

## Explicit Requirements
- Support default categories (Messages, Social, Calls, Media, System, etc.).
- Allow user to manually assign/re-assign categories.
- Provide fast text search across notification title and content.

## Out-of-scope
- On-device AI/LLM based automatic categorization (too heavy for MVP).
