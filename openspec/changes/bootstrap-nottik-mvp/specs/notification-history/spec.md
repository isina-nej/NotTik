# Notification History Specification

## Purpose
Provide a responsive, offline history of captured notifications.

## Explicit Requirements
- Must support pagination or infinite scroll.
- Must support filtering by App, Category, and Date.
- Must group by date.
- Must show a detail view with all extracted fields and revisions.
- Must support deletion of single, bulk, or all notifications.

## Out-of-scope
- Cloud syncing of history.
