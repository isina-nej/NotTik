## Requirements
### Requirement: Language
- Persian (Farsi) must be the default language.
- English must be fully supported.
- Must support full RTL for Persian and LTR for English.

### Requirement: Theme
- Must implement Light, Dark, and System themes.
- Must use an iOS-inspired, minimal, glassmorphic design system cautiously to preserve performance.

#### Scenario: Switch theme and locale
- Given the user changes settings
- Then UI updates locale and brightness accordingly
